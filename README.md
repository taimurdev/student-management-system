# Student Management System 🎓

A simple web-based Student Management System to manage student records like add, update, delete, and view students.

## Features
- Add new students
- Update student details
- Delete students
- View all records

## Tech Stack
- HTML, CSS, JavaScript
- Node.js / Express (optional backend)
- MySQL / MongoDB (optional database)

## Setup
1. Clone repo
2. Install dependencies
3. Run server
